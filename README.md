## Description

The **Banggood Products Dashboard** is an interactive data visualization tool built using **Streamlit**. It provides deep insights into Banggood product listings, helping users analyze product trends, pricing, ratings, and reviews efficiently.

This dashboard allows you to:

* Explore price distributions across different product categories.
* Identify top-rated and top-reviewed products.
* Compare rating versus price to find best value items.
* Get quick KPIs like total products, average price, average rating, and average review count.
* Filter data by categories for customized insights.

Ideal for e-commerce analysts, product managers, and anyone interested in understanding Banggood marketplace trends at a glance.
